# Remo's Roam themes

Remoa light
Add the following line to your Roam roam/css page:
'''@import url('https://github.com/remoa/roam-themes/raw/main/remoa-light.css');'''


Remoa dark
Add the following line to your Roam roam/css page:
'''@import url('https://github.com/remoa/roam-themes/raw/main/remoa-dark.css');'''
